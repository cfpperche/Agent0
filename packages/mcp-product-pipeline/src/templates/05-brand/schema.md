# Step 5 — Schema (required sections)

## Required sections

- product-name
- voice
- voice-samples
- visual-direction
- anti-patterns

## Recommended additional sections

- comparables — admired brand references with what's borrowed
- emotion-target — single emotion the user should feel in the first interaction
- color-story — color names + feelings (not codes; tokens come in step 6)

## Section content guidance

- **product-name** — final name OR shortlist with one-line rationale per candidate.
- **voice** — 1-2 paragraphs describing the voice. Concrete adjectives + what they look like in practice. Avoid "professional" / "modern" / "friendly" without examples.
- **voice-samples** — 3-5 short snippets the brand would actually say: error message, onboarding welcome, tagline, support reply, paywall. These are how downstream design/content work checks alignment.
- **visual-direction** — paragraph naming the visual feel (minimalist / playful / brutalist / etc), plus any locked decisions (typography preferences, photographic style, etc).
- **anti-patterns** — what the brand should NEVER sound like. 3-5 concrete bullets.
